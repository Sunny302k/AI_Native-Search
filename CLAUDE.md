# Về dự án

Đây là dự án kiểm thử manual và báo cáo cho dự án room booking.

Dự án room booking là 1 dự án được xây dựng trên nền tảng web với mục đích giúp các nhân viên trong công ty có thể đặt phòng họp.

# Cấu trúc thư mục

```
.
├── checklists/        # checklist QA (hiện đang rỗng)
├── docs/
│   └── specs/         # spec từng user story, 1 file/story
│       └── us07-specs.md
├── test-cases/        # tài liệu test case (hiện đang rỗng)
└── test-reports/      # báo cáo kết quả test (hiện đang rỗng)
```

# Ràng buộc

1. Chỉ thực hiện thao tác trên folder Claude, nghiêm cấm thao tác trên các folder khác.
2. Naming convention: thực hiện đặt tên file teho nguyên tắc kebap.
   1. testcase sẽ sử dụng format là CSV BOM UTF-8
   2. Check list sử dụng markdown
